# UPI Payment Trend Analysis in India

## Project Overview

This project analyzes the growth of UPI (Unified Payments Interface) digital payments in India using transaction data from 2016 onwards. The goal of this analysis is to understand the rapid adoption of digital payments and identify trends in transaction volume and value.

UPI has become one of the most widely used digital payment systems in India, enabling fast and secure transactions.

---

## Dataset

Dataset used: UPI India Monthly Transactions Dataset

The dataset contains:

- Transaction Volume (Millions)
- Transaction Value (Crores ₹)
- Monthly and yearly data
- COVID period indicator
- Seasonal indicators

---

---

## Key Analysis Performed

- Data cleaning and preprocessing
- Time series analysis of UPI transaction growth
- Yearly transaction trend analysis
- Correlation analysis between transaction volume and value
- Monthly transaction pattern analysis

---

## Visualizations

The following visualizations were created:

- UPI Transaction Volume Growth Over Time
- UPI Transaction Value Trend
- Yearly UPI Transaction Growth
- Correlation Heatmap
- Monthly Transaction Pattern

---

## Key Insights

- UPI transactions have grown exponentially since its launch in 2016.
- Digital payment usage increased significantly during the COVID-19 period.
- Transaction volume and transaction value show strong positive correlation.
- UPI has become a dominant digital payment method in India.
